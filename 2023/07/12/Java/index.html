2.1 Java Web开发基础知识

什么是struts?

Struts是一个开源的Java Web应用程序框架，用于开发基于Java的企业级Web应用程序。它提供了一套功能强大的工具和库，以简化Web应用程序的开发过程。
Struts框架遵循了MVC（Model-View-Controller）设计模式，它将应用程序的不同方面分离开来，使开发人员能够更好地组织和管理代码。
